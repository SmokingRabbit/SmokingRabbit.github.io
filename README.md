## cwfont

[cwfont](https://github.com/chowa/cwfont) for iconfont generation

### Install

```
npm i
```

### Generation iconfont

```
// cwfont compile
npm run iconfont
```

### Preview iconfont

```
// cwfont preview
npm run preview
```

### Open project

```
npm run open
```
